# RaceSync - 2026-04-29 Session Notes

## What We Fixed

### 1. Password Reset Broken (@vercel/kv → @upstash/redis)
**Problem:** User lookup failed with `ERR_MODULE_NOT_FOUND: Cannot find package '@vercel/kv'`

**Root cause:** Code was importing `@vercel/kv` but it was never in `package.json`. The code already used `@upstash/redis` style env vars (`KV_REST_API_URL`, `KV_REST_API_TOKEN`, or `UPSTASH_REDIS_REST_URL`), and `@upstash/redis` WAS in dependencies. Just wrong import.

**Files changed:**
- `api/_kv.js` — import + `automaticDeserialization: false` preserved
- `api/users/_users.js` — same
- `api/clubs/_clubs.js` — same
- `api/notifications/_notifications.js` — same

**Also fixed:** `_clubs.js` had a double `});)` syntax error introduced during edits.

### 2. Resend API Key Invalid
**Problem:** Password reset API returned 200 but email never sent. Tested directly — 401 from Resend API.

**Fix:** Chris generated a new API key at https://resend.com/api-keys and updated `RESEND_API_KEY` in Render env vars.

### 3. Added MiniMax AI as Alternative to Gemini
**Problem:** Wanted to switch from Gemini to MiniMax (already had `MINIMAX_API_KEY` set in Render).

**New files:**
- `api/ai/minimax.js` — server endpoint (OpenAI-compatible format)
  - Text: `POST https://api.minimax.io/v1/chat/completions` → `MiniMax-M2.7`
  - Image: `POST https://api.minimax.io/v1/image_generation` → `image-01` (returns base64)
- `services/minimaxService.ts` — frontend service (drop-in replacement for `geminiService`)

**Updated files:**
- `App.tsx` — imports switched from `geminiService` → `minimaxService`
- `components/AddEventModal.tsx` — same
- `components/EventDetail.tsx` — same

### 4. Render Deployment Branch Mismatch
**Problem:** Render was deploying from `migrate/upstash-redis` branch, not `main`. All the above fixes were in `main`.

**Resolution:**
- Merged all changes from `main` → `migrate/upstash-redis`
- Fixed import path in `App.tsx` (`'../services/minimaxService'` → `'./services/minimaxService'` — App.tsx is at root so path should be `./`)
- Cherry-picked the `_clubs.js` syntax fix onto `migrate/upstash-redis`
- Both branches now in sync
